# PM UPSC 2027

A modern, minimal, and distraction-free study dashboard designed to help UPSC aspirants stay organized throughout their preparation journey.

**Live Website:** https://pmupsc2027.netlify.app/

---

## Overview

PM UPSC 2027 is a personal study management platform built to track preparation, organize resources, monitor progress, and maintain consistency during the UPSC Civil Services Examination journey.

The project focuses on providing a clean interface where an aspirant can quickly access study plans, current affairs, notes, schedules, and progress without unnecessary distractions.

---

## Features

* Clean and responsive interface
* Study dashboard
* Subject-wise organization
* Daily study tracking
* Progress monitoring
* Current affairs management
* Resource organization
* Mobile-friendly design
* Fast loading performance

## Purpose

Preparing for UPSC involves managing thousands of pages of notes, current affairs, revisions, and practice tests.

This project aims to create a single digital workspace where everything required for preparation is available in one place.
